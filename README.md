Neara Multi-Vendor Marketplace App

A scalable multi-vendor marketplace platform that allows multiple vendors to sell products, manage orders, and track sales from a single system. The platform connects vendors, customers, and delivery agents to enable seamless online commerce.

This repository currently contains the Vendor App. The User App and Delivery Agent App will be added soon.

Project Overview

This project is designed to support a full marketplace ecosystem with three main applications:

Vendor App 

The Vendor App allows merchants to manage their stores and products.

Features include:

Vendor dashboard

Product management (add, edit, delete)

Order management

Sales analytics

Inventory tracking

Profile and store settings
